Initial Setup

Get the bridge running in less than two minutes.

1

Windows Server

  • 01. Run the .exe installer.
  • 02. If a "SmartScreen" popup appears, click More Info → Run Anyway.
  • 02. If a Firewall popup appears, allow both private and public networks.
2

Android Client

  • 01. Connect your android device to your PC via USB and copy the .apk file to your device.
  • 02. On your phone, open the Files app and tap the APK. If prompted, enable "Install from Unknown Sources" in settings.

Devices not connecting? Make sure both are on the same Wi-Fi network or check our Troubleshooting Guide →
